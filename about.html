Cultlang

/kʌltlæŋɡ/

Cultlang is a lisp dialect which provides an Intermediate Representation for compiler frontends. Although we may write different langauges engineers and computer scientists work every day with the same semantics. Historically when a new idea in computing was discovered, from structured programming to incremental type systems it was natural to invent along with it a new langauge to explore those ideas. Sadly this silos us, leaving programmers to evangelize new followers into their respective isolated communities.

Cult looks to change the status quo and bring a truly shared programming environment to the masses. By decoupling the Syntax of languages from their Semantics. In analogy if Cult were a compiler it would be LLVM. If it were a Star Trek technology it would be the Universal Translator. If it were a nation it would be the United Nations.

To be clear: we are talking about a machine native, incrementally typed, library rich programming environment with zero cost abstraction across syntacies that behave like your current programming environment. Additionally Cultlang's type system and runtime is embedable in your walled garden, so even if your software is reliant on your native extensions you still get access to Cult's features.
